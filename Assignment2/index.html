Day 1: (14th December)

Introduction to Web, Basics of HTML and CSS

  • Web, Architecture, Present Status
  • HTML, DOM, Manipulation
  • Flexbox
  • Grids
Day 2: (15th December)

Basics Of JavaScript

  • Revisit to the DOM
  • Browser Events and APIs- Phases of an event. Knowing when the DOM is ready
  • JS language features - Variable declaration, Promises, Use of this keyword, Rest and Spread
Day 3: (16th December)

Introduction to Frontend Development and Server

  • Modern web APIs and Asynchronous applications
  • Fetch API
  • Framework Introduction
Day 4: (17th December)

Introduction to Backend Development and Serve

  • HTTP and requests
  • Node and Express introduction
  • Application Architecture
  • Progressive Web Applications
